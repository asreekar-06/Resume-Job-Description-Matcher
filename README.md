# Resume Job Description Matcher

## 📌 Overview
AI-based web app that compares a resume with a job description and calculates an ATS score.

## 🚀 Features
- Upload resume (PDF)
- Paste job description
- Skill matching
- Missing skills detection
- ATS score calculation

## 🛠️ Tech Stack
- Python
- Streamlit
- PyPDF2
- Scikit-learn
